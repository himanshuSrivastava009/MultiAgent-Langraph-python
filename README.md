# GoRest User Management API Integration Tests

## Project Title: GoRest User Management API Integration Tests

## Description
This project provides a robust and maintainable integration test suite for the GoRest User Management API. It leverages Python, `pytest` for test execution, `Pydantic v2` for strict data modeling, `requests` for HTTP interactions, `python-dotenv` for secure configuration, and `Faker` for generating unique test data. The suite is designed to validate the API's functional behavior, error handling, and basic security aspects, ensuring adherence to the specified requirements.

## Prerequisites
Before you begin, ensure you have the following installed:
*   Python 3.9+
*   `pip` (Python package installer)
